## Installation

To build with two ways

    $ cd get_novel
    $ cargo build --release

(optional) To run unit tests:

    $ cd get_novel
    $ cargo test

(optional) To clean all except source code:

    $ cd get_novel
    $ cargo clean

## Examples

    $ cd get_novel/target/release
    $ ./get_novel -u url_novel_contents
